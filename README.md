# timeseries_prophet
This notebook explains the time series prediction on monthly car sales data . The data set was used from 
https://datamarket.com/data/set/22n4/monthly-car-sales-in-quebec-1960-1968#!ds=22n4&display=line

We will use prophet forcasting model in python to train the model for 1960 to 1968 data and then predict the outcome for 1969.
With this we were able to achive MAPE error of 7% .
